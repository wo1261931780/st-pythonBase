# 🐍 st-python.github.io

这是一个学习python语言基础知识的项目，通过 19 天的课程和代码案例，让你从头开始学习 python。

## 课程大纲

1. 标准输入输出
2. 数据类型
3. 控制语句
4. 函数和模块
5. 异常处理
6. 文件和目录
7. 正则表达式
8. 面向对象
9. 网络编程
10. 进程和线程
11. 数据库
12. GUI 编程

## 代码案例

* 利用 python 实现登录功能
* 实现文件上传功能
* 基于 TCP/IP 实现多线程聊天室
* 用 Python-MySQL 实现数据库的操作
* 用 Python 实现图形界面
